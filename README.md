# Slide In/Out Panels for Hidden Menus

Slick, space-saving side panels that slide in when you need them and disappear when you do not.

## What this is
A lightweight front-end UI pattern for hidden navigation or utility menus using smooth slide animations.

## Highlights
- Clean slide-in/slide-out interaction
- Great for mobile and compact layouts
- Simple HTML and CSS structure for easy customization

## Quick start
1. Clone the repo.
2. Open index.html in your browser.
3. Tweak styles.css to match your brand.

Build menus that feel fast, modern, and out of the way until needed.
